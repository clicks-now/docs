# Moderation

Clicks is community-run and lightly moderated.

## Who moderates

The mods team on GitHub (`clicks-now/mods`). Appointed informally — if you've been around, contributed, and want to help, ask.

## What mods do

- Review and merge content PRs
- Grant write access to contributors
- Handle code of conduct issues
- Keep the handbook and templates up to date

## What mods don't do

- Control event content or speakers
- Approve or veto city events (the host package is the standard)
- Act as a board or governing body

## Code of conduct issues

See [CODE_OF_CONDUCT.md](https://github.com/clicks-now/.github/blob/main/CODE_OF_CONDUCT.md). Raise concerns in a private Discord message to a mod.
