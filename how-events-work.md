# How events work

Each Clicks event follows the same format regardless of city.

## Steps to run an event

1. **Find a co-host** — one Shopify ecosystem partner who wants to co-run it with you
2. **Book a venue** — 40–80 capacity, central, bar or drinks available
3. **Open an event proposal** in the [events repo](https://github.com/clicks-now/events)
4. **Confirm speakers** — 3–4 people, 5 min each
5. **Promote** — Discord, LinkedIn, personal networks
6. **Run it** — use the [host package](https://github.com/clicks-now/events/tree/main/host-package)
7. **Submit a post-mortem** — PR to `cities/[city]/[YYYY-MM]/post-mortem.md`

## What the host does

- Secures the venue and co-host
- Recruits speakers
- Runs the event (or delegates to a facilitator)
- Submits the post-mortem

## What the community does

- Shows up
- Spreads the word
- Contributes back (post-mortems, content, handbook improvements)
