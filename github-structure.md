# GitHub structure

All open infrastructure for Clicks lives under the [clicks-now](https://github.com/clicks-now) org.

## Repos

| Repo | What it is |
|------|------------|
| [events](https://github.com/clicks-now/events) | Host package, run-of-show, post-mortems, event proposals |
| [content](https://github.com/clicks-now/content) | Dispatch posts, recaps, editorial |
| [docs](https://github.com/clicks-now/docs) | This handbook |
| [.github](https://github.com/clicks-now/.github) | Org profile, code of conduct, contributing guidelines |

## Access model

- **Read** — public, anyone
- **Write** — mods team and approved contributors
- **Admin** — org owners only

## Contributing

Fork any repo and open a PR. You don't need write access to contribute.

See [CONTRIBUTING.md](https://github.com/clicks-now/.github/blob/main/CONTRIBUTING.md) for details.
